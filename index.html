Description

In this example I override the original background images of the accordion component. The goal is to no longer rely on "data:image" from the original bootstrap component references because they are being blocked by a CSP directive. As an example, I replaced the original arrow icons (which used data:image) with plus and minus icons.

Sequence of steps:

  1. Include bootstrap (via download or CDN). I downloaded the official version - https://github.com/twbs/bootstrap/releases/download/v5.3.2/bootstrap-5.3.2-dist.zip (from https://getbootstrap.com/docs/5.3/ getting-started/download/)
  2. Download the svg icons you want to replace:
    1. - Go to https://icons.getbootstrap.com/
    2. - Select the desired icon, go to the "Copy HTML" section, and save this code as an svg file in your project
  3. Create a custom css file (such as bootstrap.override.css)
  4. Include this file in your project, after the line where you referenced the original bootstrap file.
  5. Make the necessary overridings according to your needs. In this example I replaced icons from the accordion component as an example.
